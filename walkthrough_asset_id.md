# Walkthrough: 各画面への資産ID表示

データベースや各種設定との紐付けを容易にするため、ダッシュボードおよび資産管理画面において各資産の固有ID（#数値）を表示するようにしました。

## 変更内容

- **ダッシュボード (`components/dashboard/category-list.tsx`)**:
  - 各資産カードの名前の左側に `#ID` を薄く表示。
- **資産管理画面 (`app/assets/page.tsx`)**:
  - 資産一覧テーブルの左端に「ID」列を追加し、資産ごとに ID を表示。

## 再現・確認手順

1. **ダッシュボードでの確認**:
   - ダッシュボード（TOPページ）を表示します。
   - 各資産カード（例：オルカンなど）の中に `#1` のようなIDが表示されていることを確認します。
2. **資産管理画面での確認**:
   - ナビゲーションやメニューから「資産管理（資産一覧）」を開きます。
   - テーブルの左側に「ID」列があり、各行に数値が表示されていることを確認します。
