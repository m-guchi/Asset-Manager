# Walkthrough: 種別デフォルト値の変更

「取引・評価を追加」ダイアログにおける「種別」のデフォルト値を、従来の「入金・投資」から「評価額更新」に変更しました。

## 変更内容

- **対象ファイル**: `app/assets/[id]/page.tsx`
- **変更箇所**:
  - `useEffect` でデータ読み込み時に設定される初期値。
  - 「取引・評価を追加」ボタンクリック時の初期化処理。
  - ダイアログが再度開かれた際の初期化処理。

## 再現・確認手順

1. **画面の表示**:
   - 資産詳細ページ（例: `/assets/1`）にアクセスします。
2. **ダイアログの起動**:
   - 「取引・評価履歴」セクションにある「取引・評価を追加」ボタンをクリックします。
3. **デフォルト値の確認**:
   - 表示されたダイアログ内の「種別」フィールドが、最初から「評価額更新」になっていることを確認します。

## 期待される結果

- 資産の種類（現金か投資資産か）に関わらず、新規追加時の「種別」のデフォルトが「評価額更新」となっていること。
