# Walkthrough: パンくずリスト（ナビゲーション）の導入

資産詳細ページにおいて、従来の戻るボタン（左矢印）を、現在の階層がひと目で分かるパンくずリスト形式のナビゲーションに変更しました。

## 変更内容

- **対象ファイル**: `app/assets/[id]/page.tsx`
- **UI変更**:
  - `ArrowLeft`（左矢印）アイコンを廃止。
  - `TOP / 親カテゴリ / 現在のカテゴリ` という形式のテキストリンクを導入。
  - 各階層（TOPおよび親カテゴリ）はクリック可能なリンクとなっており、スムーズに上位階層へ戻ることができます。
  - モダンな小文字（または大文字のアクセント）と `ChevronRight` アイコンを組み合わせたプレミアムなデザインに調整しました。

## 再現・確認手順

1. **画面の表示**:
   - 子カテゴリを持つ資産の詳細ページ（例: `旧NISA オルカン` など）を開きます。
2. **ナビゲーションの確認**:
   - 画面上部に `TOP / オルカン / 旧NISA オルカン` のように表示されていることを確認します。
3. **リンクの動作確認**:
   - `TOP` をクリックするとダッシュボード（ルート）に戻ること。
   - 親カテゴリ名（例: `オルカン`）をクリックすると、その親カテゴリの詳細ページに移動すること。
