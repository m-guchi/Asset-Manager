# Walkthrough: 資産詳細画面および管理画面へのID表示

資産IDの表示箇所を整理し、ダッシュボードでは非表示、詳細表示時および管理画面のみに表示するように変更しました。

## 変更内容

- **パンくずリストへのID追加 (`app/assets/[id]/page.tsx`)**:
  - 資産名をクリックした後の詳細ページ上部、ナビゲーションの右端に `#ID` を薄く表示するようにしました。
- **ダッシュボードからの削除 (`components/dashboard/category-list.tsx`)**:
  - TOP画面の資産カードからは ID 表示を削除し、以前のクリーンな表示に戻しました。
- **資産管理画面 (`app/assets/page.tsx`)**:
  - こちらは管理上必要なため、引き続き「ID」列を表示しています。

## 再現・確認手順

1. **ダッシュボードの確認**:
   - TOPページを表示します。
   - 資産カードから `#ID` が消え、元のデザインに戻っていることを確認します。
2. **詳細画面での確認**:
   - 任意の資産カードをクリックして詳細ページを開きます。
   - 画面左上のパンくずリスト（TOP / ... / 資産名）の右側に、控えめに `#1` のようなIDが表示されていることを確認します。
3. **資産管理画面での確認**:
   - 「資産管理」画面では、引き続きテーブルの左端にIDが表示されていることを確認します。
